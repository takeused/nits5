# UI 디자인 점검 기록

## 2026-07-13

- 검색·분석·연구비 화면의 색상, 테두리, 모서리, 그림자, 포커스 표현을 공통 CSS 토큰으로 정리했다.
- 기본 캔버스는 밝은 중성 배경으로 통일하고, 인디고 계열은 선택·강조 상태에만 사용하도록 했다.
- 헤더, 검색창, DB 전환, 탭, ScienceON/NTIS 표, 분석 카드, 모달, 페이지네이션, 토스트에 동일한 radius/elevation/focus 규칙을 적용했다.
- 모바일 폭에서 히어로·검색창·표 컨테이너의 밀도를 낮추는 반응형 보정을 추가했다.
- 기존 동작을 보존하는 CSS 레이어로 적용했으며, 다음 단계는 실제 브라우저에서 홈/검색/분석/다크모드 상태를 시각 점검하는 것이다.
